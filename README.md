# AppStyleGuides

Este proyecto está construido con [Angular CLI](https://github.com/angular/angular-cli) version 10.0.3. [Angular Material](https://material.angular.io/) version 10.0.2 para componentes UI y [Bootstrap](https://getbootstrap.com/) version 4.5.0 para el layout.

También tiene una capa adicional de estilos personalizados con [SASS](https://sass-lang.com/) utilizando la arquitectura [SMACSS](http://smacss.com/) y la metodología [BEEM](http://getbem.com/).
