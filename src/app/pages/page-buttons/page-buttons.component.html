Para dar estilo al botón, agregamos el elemento button y añadimos el atributo mat-button y la clase .button. 

<button mat-button class="button">Button</button>

Alineado y espaciado

Si queremos añadir alineaciones y separaciones, utilizamos las clases de margin, padding y align de Bootstrap.

Para añadir separaciones podemos utilizar la clase .mr- con la medida de separación [ 0-5 ] para generar un márgen a la derecha.
Del mismo modo, si queremos separaciones para los demás lados, utilizamos .ml-, .mt-, .mb-, etc.
Para mayor referencia, consulte la guía de espaciados en la documentación oficial de Bootstrap. 

<button mat-button class="button mr-2">Two spaces</button>
<button mat-button class="button mr-3">Three spaces</button>
<button mat-button class="button mr-4">Four spaces</button>
<button mat-button class="button mr-5">Five spaces</button>

También para las alineaciones utilizamos la clase .justify-content- con sus valores [ start, center, end, between, around ] para horizontal y .align-self- [ start, center, end ] para vertical.
Hay que tener en cuenta que en este caso debemos utilizar un elemento contenedor con la clase .d-flex. Para mayor referencia, consulte la guía de alineado en la documentación oficial de Bootstrap. 

<p class="d-flex justify-content-around p-3">
    <button mat-button class="button color-accent align-self-start">Button start</button>
    <button mat-button class="button color-accent align-self-center">Button center</button>
    <button mat-button class="button color-accent align-self-end">Button end</button>
</p>

Botones de colores

Use las clases .color-first .color-accent .color-dark .color-light .color-cancel .color-success .color-danger para obtener botones con los colores coorporativos.
Por defecto el color es celeste 

<button mat-button class="button">Color default</button>
<button mat-button class="button color-first">Color third</button>
<button mat-button class="button color-accent">Color accent</button>
<button mat-button class="button color-dark">Color dark</button>
<button mat-button class="button color-light">Color light</button>
<button mat-button class="button color-cancel">Color cancel</button>
<button mat-button class="button color-success">Color success</button>
<button mat-button class="button color-danger">Color danger</button>

Botones transparentes

Los botones transparentes solo tiene color en el borde y en el texto. Para utilizarlos, añada la clase .ghost. 

<button mat-button class="button ghost">Button ghost</button>

Puede añadir colores utilizando las clases anteriormente descritas .color-first .color-accent .color-dark .color-light .color-cancel .color-success .color-danger 

<button mat-button class="button ghost">Ghost Default</button>
<button mat-button class="button ghost color-first">Ghost third</button>
<button mat-button class="button ghost color-accent">Ghost accent</button>
<button mat-button class="button ghost color-dark">Ghost dark</button>
<button mat-button class="button ghost color-light">Ghost light</button>
<button mat-button class="button ghost color-cancel">Ghost cancel</button>
<button mat-button class="button ghost color-success">Ghost success</button>
<button mat-button class="button ghost color-danger">Ghost danger</button>

Tamaños

Puede añadir las clases .biggest .bigger .big .small .smaller .nano para utilizar diferentes tamaños de botón.
Si no añade ninguna clase, por defecto utiliza el tamaño normal. 

<button mat-button class="button biggest">Biggest</button>
<button mat-button class="button bigger">Bigger</button>
<button mat-button class="button big">Big</button>
<button mat-button class="button">Normal</button>
<button mat-button class="button small">Small</button>
<button mat-button class="button smaller">Smaller</button>
<button mat-button class="button nano">Nano</button>

Si desea que el botón ocupe todo el ancho disponible, añada la clase .full 

<button mat-button class="button full">Full</button>

Botón desactivado

Añada el atributo disabled para obtener un botón desactivado. 

<button mat-button class="button" disabled>Button Disabled</button>
