Familias Tipográficas

Se utiliza dos familias tipográficas:

  • Red Hat Display (400, 500, 700) Para los títulos.
  • Roboto (300, 400, 700) Para los contenidos.

Estas tipografías se encuentra definidas en las variables CSS --heading-font y --body-font.
Utilice las clases .heading-font y .body-font para aplicar estas fuentes a cualquier elemento. 

:root {
    // Font families
    $heading-font : 'Red Hat Display', sans-serif;
    $body-font    : 'Roboto', sans-serif;

    --heading-font: #{$heading-font};
    --body-font: #{$body-font};
}
.heading-font {
    font-family: var(--heading-font);
}
.body-font {
    font-family: var(--body-font);
}

Tamaños de fuentes

Las siguientes variables se utilizan para definir el tamaño de los textos. Estos tamaños cambian a partir del breakpoint lg. 

:root {
    $base  : 16px;
    $ratio : 1.2;
    
    $title3  : $base * $ratio;
    $title2  : $title3 * $ratio;
    $title1  : $title2 * $ratio;
    $bigger-title : $title1 * $ratio;
    $small   : $base / $ratio;
    $smaller : $small / $ratio;
    $nano    : $smaller / $ratio;
    
    // Font sizes
    --bigger-font-size    :  #{$title1};
    --h1-font-size        :  #{$title2};
    --h2-font-size        :  #{$title3};
    --h3-font-size        :  #{$base};
    --normal-font-size    :  #{$small};
    --small-font-size     :  #{$smaller};
    --smaller-font-size   :  #{$nano};
    --nano-font-size      :  #{$nano};
    --heading-line-height : 1.3;
    --body-line-height    : 1.6;

    // Font sizes a partir del breakpoint lg (64em ~ 1024px)
    @include from(lg) { 
        --bigger-font-size    :  #{$bigger-title};
        --h1-font-size        :  #{$title1};
        --h2-font-size        :  #{$title2};
        --h3-font-size        :  #{$title3};
        --normal-font-size    :  #{$base};
        --small-font-size     :  #{$small};
        --smaller-font-size   :  #{$smaller};
        --nano-font-size      :  #{$nano};
    }
}

Regular, Negrita, Ligera y Cursiva

Por defecto el peso tipográfico de las fuentes es regular, sin embargo puede usar las clases .bold para negrita, .light-text para ligera o .italic para cursiva.
Los títulos del <h1> al <h3> por defecto son negita, del <h4> al <h6> son regular, pero si quiere que el peso sea aún menor, utilice la clase .light-title
